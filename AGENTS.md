# AGENTS.md — NixOS flake

## Rebuild

```sh
sudo nixos-rebuild switch --flake '.#HOST'   # HOST = folder name under system
nix-collect-garbage -d                        # user gc
sudo /run/current-system/bin/switch-to-configuration boot  # boot after gc
```

## Host system layout

```
system/<hostname>/
├── config.nix         # { dpi, system, kbLayout, bootPath } - consumed by flake.nix
├── hardware.nix       # generated by nixos-generate-config (do not edit manually)
└── configuration.nix  # per-host config
```

Host names: `pc`, `t14`, `xps15`. Each is auto-discovered from `system/` subdirectory names.

## Secrets

- `secrets/*.json` are **git-crypt encrypted** (`.gitattributes` defines the filter).
- Loaded in `flake.nix` via `builtins.fromJSON` — each file becomes a top-level key.
- Contains `user.json` (name + password hash). Username is `binh`.
- An agent cannot read them without `git-crypt unlock`.

## Structure

- **`configuration.nix`** — system entry point; auto-imports all `core/*.nix`
- **`core/`** — shared system modules (font, session, system, user)
- **`modules/`** — optional feature modules: `8bitdo`, `game`, `nordvpn`, `qudelix`, `uutils`
- **`overlays/`** — auto-imported overlay *.nix files (chromium, common, rofi)
- **`home/`** — home-manager config (shared across hosts via `home/home.nix`); each file is a module
- **`system/<hostname>/`** — per-host config as described above

## Desktop / display

- SDDM with Wayland, auto-login as `binh`, default session `mango`
- Mango is the compositor. DMS (DankMaterialShell) provides widgets (system monitor, VPN, audio visualizer, calendar, clipboard via `wtype`).
- Fcitx5 with Unikey for Vietnamese input.

## Noteworthy config

- `stateVersion = "25.11"` (set in `flake.nix`)
- `nix.settings.experimental-features = ["nix-command" "flakes"]` (in `configuration.nix`)
- Firewall disabled (`networking.firewall.enable = false`)
- NextDNS over TLS via `services.resolved`
- Docker data root at `/docker`
- IWD WiFi backend
- `uutils` module replaces coreutils/findutils/diffutils with Rust equivalents

## Dev shell

```sh
direnv allow   # loads shell.nix via .envrc (use nix)
```
`shell.nix` provides `nixd` (Nix LSP), `lua-language-server`, `haskell-language-server`.
